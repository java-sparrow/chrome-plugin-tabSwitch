# Chrome 插件： tab切换
使用鼠标右键+滚动滚轮，可以切换同一窗口的不同标签页

## 版本：0.1
## 更新日志
  * 2016-03-07 基本功能完成，支持在已激活的标签页上 使用鼠标右键+滚动滚轮 切换标签
